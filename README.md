# dsa-patterns-js

A structured collection of **Data Structures & Algorithms (DSA)** implemented in **JavaScript**, organized by common **interview patterns** such as Binary Search, Two Pointers, Fast & Slow Pointers, Sliding Window, Prefix Sums, and more.

This repo is designed as a **pattern-first roadmap** to mastering problem-solving for coding interviews. Each folder corresponds to a well-known DSA pattern and contains:
- ✅ Clean, tested JavaScript solutions
- ✅ Problem statements & approaches
- ✅ Time & space complexity notes
- ✅ Edge cases and variations

Perfect for practicing, revising, or showcasing your problem-solving skills in interviews.
